## `GET` Get all product variations

```http
https://api.moltin.com/v2/variations
```

<!--DOCUSAURUS_CODE_TABS-->

<!-- Request -->

**Path Parameters**:

| Name | Required | Type | Description |
| --- | --- | --- | --- |
| `Authorization` | Required | `string` | The Bearer token required to get access to the API. For `grant_type`, you must use `client_credentials`. |
