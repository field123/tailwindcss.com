Elastic Path provides a multilingual service for the [Product API](../../catalog/products/index.md). Using the [Settings API](../settings/update-settings.md#put-update-project-settings), you can add additional supported languages. When you [update a product](../../catalog/products/update-a-product.md), you can specify the `X-Moltin-Language` header.

:::note
You can create translations for both product `name` and `description` fields.
:::
