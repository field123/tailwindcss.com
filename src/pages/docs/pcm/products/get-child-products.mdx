## `GET` Get Child Products

```http
https://api.moltin.com/pcm/products/:productId/children
```

## Parameters

### Path parameters

| Name | Required | Type | Description |
| --- | --- | --- | --- |
| `productId` | Required | `string` | The ID of the base product to list child products for. |

### Headers

| Name | Required | Type | Description |
| --- | --- | --- | --- |
| `Authorization` | Required | `string` | The Bearer token required to get access to the API. |

## Request Example

```bash
curl GET https://api.moltin.com/pcm/products/b9ea1ff1-8e24-4d60-a6dd-5bb39fb25aa0/children \
     -H "Authorization: Bearer XXXX" \
     -H "Content-Type: application/json" \
```
